# exp-share

Custom Claude Code skills, shared across projects.

## Skills

| Skill | Description |
|-------|-------------|
| `/commit` | Stage, commit, and push changes as semantically grouped commits |

## Setup

Symlink the `skills/` directory into your workspace's `.claude/` folder:

```sh
ln -s /path/to/exp-share/skills /path/to/workspace/.claude/skills
```
